# flashlight
Фонарик управляемый командами с сервера
## Инструкция по установке
1. Клонировать репозиторий ```git clone https://github.com/voreoneks/flashlight.git```
2. Создать виртуальное окружение (использовать virtualenv)
3. Установить зависимости ```pip3 install -r requirements.txt```
4. Запустить проект ```make run```
5. Зайти на ```http://localhost:8000/docs```
